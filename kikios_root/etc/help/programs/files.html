File Manager

Browse and manage files on disk.

Features

  • Directory browsing
  • File icons by type
  • Double-click to open
  • Navigate with path bar

Usage

  • Double-click folders to enter
  • Double-click files to open
  • Click path segments to navigate up

File Types

  • Folders - Enter directory
  • .txt - Open in TextEdit
  • .py - Run with MicroPython
  • .mp3/.wav - Play in Music
  • Other - Open in viewer

Launch

Click Files icon in dock, or: /bin/files
