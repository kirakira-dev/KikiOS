MicroPython

Python 3 interpreter for VibeOS. Run Python scripts or use the interactive REPL.

Usage

micropython - Start interactive REPL
micropython script.py - Run a script
micropython script.py arg1 arg2 - Run with arguments

VibeOS Module

import vibe for system access. See Python API for details.

Shebang

Make executable Python scripts:
#!/bin/micropython
print("Hello!")

Limitations

MicroPython, not full CPython.
Limited standard library.
No pip or package manager.

Launch

/bin/micropython
