vim - Text Editor

Modal text editor (vi clone) for terminal use.

Modes

Normal: Navigate and commands
Insert: Type text (press i)
Command: Save/quit (press :)

Normal Mode

h/j/k/l: Move cursor
i: Enter insert mode
x: Delete character
dd: Delete line
yy: Yank (copy) line
p: Paste
u: Undo
/: Search

Commands

:w - Save
:q - Quit
:wq - Save and quit
:q! - Quit without saving

Launch

/bin/vim [file]
