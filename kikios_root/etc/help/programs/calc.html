Calculator

Desktop calculator with floating-point support.

Features

  • Basic operations: + - * /
  • Decimal point support
  • Keyboard input

Usage

  • Click buttons or use keyboard
  • Press C to clear
  • Press = or Enter for result
  • Press Q to quit

Keyboard

0-9: Number input
.: Decimal point
+-*/: Operations
=: Calculate result
c/C: Clear
q/Q: Quit

Launch

Click Calculator icon in dock, or: /bin/calc
