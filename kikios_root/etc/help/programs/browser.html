Browser

Web browser with HTTP and HTTPS support.

Features

  • HTTP and HTTPS (TLS 1.2)
  • Basic HTML rendering
  • Links, headings, lists, paragraphs
  • Mouse wheel scrolling
  • Local file:// URLs

Usage

  • Enter URL in address bar, press Enter
  • Click links to navigate
  • Back button for history
  • Refresh button to reload

Supported URLs

http://example.com
https://google.com
file:///etc/help/index.html
about:home

Launch

Click Browser icon in dock, or: /bin/browser [url]
