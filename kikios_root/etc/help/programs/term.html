Terminal

Terminal emulator for running shell commands.

Features

  • Runs kikish (VibeOS shell)
  • Scrollback buffer
  • Mouse wheel scrolling

Usage

  • Type commands and press Enter
  • Use arrow keys for history
  • Scroll with mouse wheel

Example Commands

ls /bin
cat /etc/motd
ping 1.1.1.1
micropython

Launch

Click Terminal icon in dock, or: /bin/term
