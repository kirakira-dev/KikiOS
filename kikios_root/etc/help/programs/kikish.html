kikish - VibeOS Shell

The default command-line shell for VibeOS.

Features

  • POSIX-like command syntax
  • Command history (arrow keys)
  • Output redirection (> and >>)
  • Environment variables
  • Tab completion

Built-in Commands

cd, pwd, exit, export, echo

Redirection

echo "hello" > file.txt
echo "more" >> file.txt
cat < file.txt

Launch

Runs automatically in Terminal. Direct: /bin/kikish
