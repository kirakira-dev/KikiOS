KikiCode

Code editor with syntax highlighting.

Features

  • Syntax highlighting for C and Python
  • Line numbers
  • File browser sidebar
  • Save and load files

Usage

  • Browse files in left sidebar
  • Click to open file
  • Edit code in main area
  • Save with keyboard shortcut

Launch

Click KikiCode icon in dock, or: /bin/kikicode [file]
