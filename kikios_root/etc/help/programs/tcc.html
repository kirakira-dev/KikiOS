TCC - Tiny C Compiler

C compiler that runs natively on VibeOS. Compile and run C programs without leaving the OS.

Usage

tcc -run file.c - Compile and run immediately
tcc -o prog file.c - Compile to executable
tcc -c file.c - Compile to object file

Include Path

System headers in /lib/tcc/include/
Includes kiki.h for VibeOS API access.

Example

#include <kiki.h>
int main(kapi_t *k, int argc, char **argv) {
    k->puts("Hello from TCC!\\n");
    return 0;
}

Features

  • Fast compilation
  • Run directly without separate link step
  • C99 support
  • Small binary size

Launch

/bin/tcc [options] file.c
