TextEdit

Simple text editor for creating and editing files.

Features

  • Create and edit text files
  • Save with File > Save As
  • Basic text editing

Usage

  • Type to enter text
  • Arrow keys to navigate
  • Backspace to delete
  • File > Save As to save

Keyboard

Arrow keys   Navigate
Backspace    Delete character
Enter        New line

Launch

Click TextEdit icon in dock, or: /bin/textedit [file]
