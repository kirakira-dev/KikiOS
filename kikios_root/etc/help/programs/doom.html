DOOM

The classic 1993 first-person shooter, ported to VibeOS.

Requirements

Requires DOOM1.WAD (shareware) or DOOM.WAD in /doom directory.

Controls

Arrow keys: Move/turn
Ctrl: Fire
Space: Use/open doors
Shift: Run
1-7: Select weapon
Tab: Map
Esc: Menu

Notes

Runs at 35 FPS (original game speed).
Sound supported on QEMU.

Launch

/bin/doom
