Desktop

The window manager and desktop environment.

Features

  • Menu bar with Apple, File, Edit menus
  • Dock at bottom for launching apps
  • Draggable windows with close buttons
  • Mouse cursor

Usage

  • Click dock icons to launch apps
  • Drag window title bars to move
  • Click close box (top-left) to close
  • Apple menu: About VibeOS, Quit

Launch

Starts automatically at boot. Run manually: /bin/desktop
