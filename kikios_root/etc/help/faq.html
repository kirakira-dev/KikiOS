Frequently Asked Questions

What hardware does VibeOS run on?

QEMU virt machine (aarch64) and Raspberry Pi Zero 2W.

Can I run Linux programs?

No. Programs must be compiled for VibeOS using the VibeOS API.

Does VibeOS have memory protection?

No. Flat memory model, no MMU. All code runs in kernel space.

How does multitasking work?

Cooperative. Programs call yield() to give up the CPU.

How do I transfer files?

Mount disk.img on your host (hdiutil attach disk.img on macOS) and copy files. Unmount before running QEMU.

Can I write GUI apps in Python?

Yes! See the Python API. The browser is written in Python.

Why black and white graphics?

Aesthetic choice for retro Mac System 7 vibes.
