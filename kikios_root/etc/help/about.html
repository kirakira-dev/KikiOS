About VibeOS

VibeOS is a hobby operating system built from scratch for ARM64 (aarch64), targeting QEMU and Raspberry Pi Zero 2W.

Design Philosophy

  • Retro aesthetic inspired by classic Mac System 7
  • Simple, educational, nostalgic
  • Monolithic kernel (like Windows 3.1)
  • Cooperative multitasking

Features

  • Graphical desktop with windows, menus, dock
  • FAT32 filesystem with persistent storage
  • Networking (Ethernet, TCP/UDP, DNS)
  • HTTPS via TLS 1.2
  • Audio playback (WAV, MP3)
  • Python scripting via MicroPython
