Console I/O

void putc(char c)

Print a single character.

void puts(const char *s)

Print a string.

void uart_puts(const char *s)

Print directly to UART (serial console).

int getc(void)

Read a character. Non-blocking, returns -1 if no input.

int has_key(void)

Check if keyboard input is available.

void set_color(uint32_t fg, uint32_t bg)

Set foreground and background colors (0x00RRGGBB).

void clear(void)

Clear the entire screen.

void set_cursor(int row, int col)

Move cursor to row, col position.

void set_cursor_enabled(int enabled)

Show (1) or hide (0) the text cursor.

void clear_to_eol(void)

Clear from cursor to end of line.

int console_rows(void), int console_cols(void)

Get console dimensions in characters.
