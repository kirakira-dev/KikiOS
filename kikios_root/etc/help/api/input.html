Input

Keyboard

int has_key(void)

Check if keyboard input is available.

int getc(void)

Read a character. Non-blocking, returns -1 if no input.

Mouse

void mouse_get_pos(int *x, int *y)

Get current mouse position on screen.

uint8_t mouse_get_buttons(void)

Get button state (bit 0 = left, bit 1 = right, bit 2 = middle).

void mouse_poll(void)

Poll for mouse updates.

void mouse_set_pos(int x, int y)

Set mouse position.

void mouse_get_delta(int *dx, int *dy)

Get and clear accumulated movement delta.

Stdio Hooks

Terminal emulators can redirect I/O:

stdio_putc, stdio_puts, stdio_getc, stdio_has_key

If set, shell uses these instead of console I/O.
