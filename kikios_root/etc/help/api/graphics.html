Graphics

Framebuffer

uint32_t *fb_base

Direct pointer to framebuffer. Pixels are 0x00RRGGBB.

uint32_t fb_width, fb_height

Screen dimensions in pixels.

void fb_put_pixel(uint32_t x, uint32_t y, uint32_t color)

Draw a single pixel.

void fb_fill_rect(uint32_t x, uint32_t y, uint32_t w, uint32_t h, uint32_t c)

Fill a rectangle.

void fb_draw_char(uint32_t x, uint32_t y, char c, uint32_t fg, uint32_t bg)

Draw a character using the bitmap font.

void fb_draw_string(uint32_t x, uint32_t y, const char *s, uint32_t fg, uint32_t bg)

Draw a string.

Font

const uint8_t *font_data

8x16 bitmap font (256 chars, 16 bytes each).

Note

For window apps, use window_get_buffer() instead of fb_base. Include gfx.h for helper functions.
