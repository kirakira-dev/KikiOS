Process

void exit(int status)

Exit the current process with status code.

int exec(const char *path)

Run another program and wait for it to finish.

int exec_args(const char *path, int argc, char **argv)

Run program with arguments, wait for completion.

void yield(void)

Give up CPU to other processes. Call this in loops.

int spawn(const char *path)

Start a new process (returns immediately).

int spawn_args(const char *path, int argc, char **argv)

Spawn process with arguments.

void sleep_ms(uint32_t ms)

Sleep for at least ms milliseconds.

int kill_process(int pid)

Kill a process by PID.

int get_process_count(void)

Get number of active processes.

int get_process_info(int index, char *name, int size, int *state)

Get process name and state by index.
