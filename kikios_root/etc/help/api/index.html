C API Reference

Programs receive a kapi_t pointer with kernel functions. Include kiki.h and use k->function().

Categories

  • Console I/O - putc, puts, getc, colors, cursor
  • Memory - malloc, free
  • Filesystem - open, read, write, mkdir, readdir
  • Process - exit, exec, spawn, yield, sleep
  • Window - window_create, events, buffers
  • Graphics - framebuffer, font, pixel drawing
  • Input - mouse position, buttons, keyboard
  • Network - TCP, TLS, DNS, ping
  • Audio - sound playback, pause, resume
  • System - time, RTC, memory info, process info
  • Hardware - DMA, GPIO LED, USB, kernel log

Example

int main(kapi_t *k, int argc, char **argv) {
    k->puts("Hello, VibeOS!\\n");
    return 0;
}
