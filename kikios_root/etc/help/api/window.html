Window Management

int window_create(int x, int y, int w, int h, const char *title)

Create a window. Returns window ID or -1 on failure.

void window_destroy(int wid)

Close and destroy a window.

uint32_t *window_get_buffer(int wid, int *w, int *h)

Get pixel buffer for drawing. Sets w/h to content dimensions.

void window_invalidate(int wid)

Mark window as needing redraw.

void window_set_title(int wid, const char *title)

Change window title.

int window_poll_event(int wid, int *type, int *d1, int *d2, int *d3)

Poll for window events. Returns 1 if event available.

Event Types

WIN_EVENT_MOUSE_DOWN (1) - d1=x, d2=y
WIN_EVENT_MOUSE_UP (2) - d1=x, d2=y
WIN_EVENT_MOUSE_MOVE (3) - d1=x, d2=y
WIN_EVENT_KEY (4) - d3=keycode
WIN_EVENT_CLOSE (5) - close button clicked
WIN_EVENT_RESIZE (8) - window resized
