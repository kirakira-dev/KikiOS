Memory

void *malloc(size_t size)

Allocate size bytes of memory. Returns NULL on failure.

void free(void *ptr)

Free previously allocated memory.

Memory Info

size_t get_mem_used(void)

Get used heap memory in bytes.

size_t get_mem_free(void)

Get free heap memory in bytes.

size_t get_ram_total(void)

Get total RAM in bytes.

Example

char *buf = k->malloc(1024);
if (buf) {
    // use buffer
    k->free(buf);
}
