VibeOS is a hobby operating system built from scratch for the ARM64 architecture. It was created as an experiment to see what an LLM (Large Language Model) could build when paired with a human who provides "vibes only".

History

VibeOS development began in 2024 as a collaboration between an AI assistant (Claude) and a human developer. The project follows a unique development methodology where the human provides creative direction and enthusiasm ("vibes") while Claude handles all technical implementation decisions.[1]

"This is a science experiment to see what an LLM can build." — Project README

Features

VibeOS includes a variety of features typical of a hobby operating system:

  • GUI Desktop – A graphical desktop environment inspired by classic Mac System 7
  • File System – FAT32 support with long filename (LFN) capability
  • Networking – TCP/IP stack with DNS resolution
  • Web Browser – Basic HTML rendering with HTTPS support
  • Audio – WAV and MP3 playback via virtio-sound
  • Applications – Calculator, text editor, file manager, music player

Technical Details

Architecture

VibeOS runs on the AArch64 (ARM64) architecture, specifically targeting QEMU's virt machine with a Cortex-A72 CPU.

Memory Model

The operating system uses a flat memory model without virtual memory protection, similar to Windows 3.1. All processes share the same address space.

Memory Layout:
0x40000000 - DTB (Device Tree)
0x40200000 - Kernel
0x41000000 - Programs
0x4F000000 - Stack
